¿Qué es la clave SSH?

Se trata de una clave que nos permite dar un paso en la protección de nuestros proyectos en GitHub. Usando el protocolo SSH, te puedes conectar y autenticar con servicios y servidores remotos. Con las llaves SSH puedes conectarte a GitHub sin proporcionar tu nombre de usuario y token de acceso personal en cada visita.

¿Tengo que configurarla por única vez?

Vas a necesitar configurarla por única vez siempre que no tengas que formatear la pc. Tendrás que seguir los pasos con cada dispositivo que pongas a disposición para trabajar con código y quieras subirlo a GitHub de forma segura.

Pasito a Pasito

1. Abri la consola de GIT BASH

2. Ingresá el comando, incluye el correo electronico que tenes vinculado a GitHub:

ssh-keygen -t ed25519 -C "tu-email@gmail.com"

3. Es esperable ver: Generating public/private ed25519 key pair.

Presionar ENTER

4. Es esperable ver: Enter file in which to save the key (/c/Users/Naty/.ssh/id_ed25519): Created directory '/c/Users/Naty/.ssh'.

Presionar ENTER

5. Es esperable ver: Enter passphrase (empty for no passphrase):

Presionar ENTER

6. Es esperable ver: Enter same passphrase again:

Presionar ENTER

7. Es esperable ver:
Your identification has been saved in /c/Users/Naty/.ssh/id_ed25519 Your public key has been saved in /c/Users/Naty/.ssh/id_ed25519.pub The key fingerprint is: SHA256:1++TPAVbn61tJmCyoYVPBoiybpBU5/nyxmL3+VwPM6cPK1E tu-emailo@gmail.com The key's randomart image is: +--[ED25519 256]--+ | ..o. . | | o o . | | . o | | . o I .| | Soo o. ++| | ..*+-...+| | *o=+*=o| | ..o+.oB@*| | .*O=| +----[SHA256]-----+ Ingresar el siguiente comando:

eval "$(ssh-agent -s)"

8. Es esperable ver: Agent pid 123
Ingresar el siguiente comando:

ssh-add ~/.ssh/id_ed25519

9. Es esperable ver: Identity added: /c/Users/Naty/.ssh/id_ed25519 (tu-email@gmail.com)
Ingresá el siguiente comando, es el que va a copiar la clave para pegarla en GitHub:

clip < ~/.ssh/id_ed25519.pub

10. Ir a la web de GitHub, iniciar sesión e ir a Settings

11. En la barra lateral de configuración de usuario, da clic en Llaves SSH y GPG.

12. Click en New SSH key (Nueva clave SSH) o Add SSH key (Agregar clave SSH).

13. En el input Title ingresá un nombre que identifique la compu donde estas generando la clave. Ejemplo: "Notebook Lenovo del trabajo"
En el input Key copia la clave que trajiste desde Git

14. Click en Add SSH key (Agregar tecla SSH).

15. Si te lo pide confirmá tu contraseña de GitHub
